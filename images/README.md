# Bilder-Ordner für Up de Warft

Hier uploadest du deine Fotos der Ferienwohnung.

## Benötigte Bilder:

1. **hero.jpg** - Titelbild (wird oben groß angezeigt)
2. **wohnzimmer.jpg** - Wohnzimmer
3. **schlafzimmer1.jpg** - Schlafzimmer 1
4. **schlafzimmer2.jpg** - Schlafzimmer 2
5. **kueche.jpg** - Küche
6. **badezimmer.jpg** - Badezimmer
7. **terrasse.jpg** - Terrasse & Garten

## Wie du Bilder hochlädst:

1. Gehe auf GitHub zu deinem Repo
2. Klick auf den **images** Ordner
3. Klick "Add file" → "Upload files"
4. Wähle deine JPG/PNG Bilder
5. Speichern (Commit)

Die Bilder sollten nicht größer als 2-3 MB sein für schnelle Ladezeiten.
